## 🚀 Deployment

1. **Model Setup:**
   ```bash
   ollama create gemma4:unrestricted -f Unrestricted_Gemma4.Modelfile
   ```

2. **Run Environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate
   pip install -r requirements.txt
   ```

3. **Execution:**
   ```bash
   python src/main.py
   ```
